=========
Geostates
=========

.. image:: https://img.shields.io/pypi/v/geostates.svg
   :target: https://pypi.org/project/geostates

.. image:: https://img.shields.io/badge/License-MIT-lightgray.svg?style=flt-square
   :target: https://opensource.org/licenses/MIT


Geostates is a Python 3.6+ pacakge built to obviate the difficulties associated with constructing choropleth plots of the United States. This package is designed to augment shapefile visualizations by placing non-contiguous regions in thier canonical repositioned locations. This page includes information on how to use the package, its full API, examples, reference resources, and contribution guidlines.

If you encounter any problems with the package, please feel free to file an issue on the issue tracker: https://github.com/eolesinski/geostates/issues
